<task type="auto">
  <name>Task 1: Criar vercel.json</name>
  <files>vercel.json</files>
  <action>
    ```json
    {
      "$schema": "https://openapi.vercel.sh/vercel.json",
      "rewrites": [
        { "source": "/((?!api/.*).*)", "destination": "/index.html" }
      ]
    }
    ```

    Nota: o regex `/((?!api/.*).*)` permite que rotas tipo `/api/...` passem sem rewrite (se um dia tivermos Vercel Functions). Como hoje só temos SPA, qualquer rota cai em index.html.

    Avoid:
    - Adicionar `buildCommand` / `outputDirectory` — Vercel detecta Vite sozinho e adiciona ruído desnecessário
    - Adicionar `framework: "vite"` explicitamente — também auto-detectado
    - Headers customizados — sem necessidade no MVP
  </action>
  <verify>Arquivo existe na raiz; JSON válido; build local continua passando.</verify>
  <done>AC-1 satisfeito.</done>
</task>
