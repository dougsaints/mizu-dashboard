## Goal
Portar 2 visualizações de produtos delivery (Anota AI) do painel HTML pro React:
1. **Donut por categoria** (HTML linha 4374-4404, `renderCategoryDonut`) — participação % de cada categoria de produto no total vendido.

- Donut responde "como o cardápio se distribui?" — controle do mix entre Sushi/Hot/Bebida/etc.
- Ambos completam a parte de produtos que o `TopProductsChart` atual só cobre como ranking top 10.

    ```ts
    // Mapeamento heurístico (case-insensitive, lowercase + remove acentos)
    const HEURISTIC: Array<[RegExp, string]> = [
      [/sushi|sashimi|nigiri|temaki/i, '#E74C3C'],   // vermelho-sushi
      [/hot[\s_-]*roll|empanad/i,       '#E67E22'],   // laranja-hot
      [/combinado|combo|cmb/i,          '#8E44AD'],   // roxo-combinado
      [/bebida|drink|suco|refri/i,      '#3498DB'],   // azul-bebida
      [/sobremes|dessert|doce/i,        '#F39C12'],   // âmbar-sobremesa
      [/entrada|aperitivo|petisc/i,     '#16A085'],   // verde-entrada
    ]

    // Paleta de fallback determinística (hash do nome → índice)
    const FALLBACK = ['#95A5A6', '#7F8C8D', '#34495E', '#BDC3C7', '#1ABC9C', '#9B59B6', '#27AE60', '#D35400']

    function hashCategory(s: string): number {
      let h = 0
      for (let i = 0; i < s.length; i++) {
        h = (h * 31 + s.charCodeAt(i)) | 0
      }
      return Math.abs(h)
    }

    export function categoryColor(category: string | null | undefined): string {
      if (!category || !category.trim()) return '#A89E89' // sem categoria: cinza
      for (const [re, color] of HEURISTIC) {
        if (re.test(category)) return color
      }
      return FALLBACK[hashCategory(category.toLowerCase()) % FALLBACK.length]
    }

    // Adiciona transparência hex (ex: "CC" = 80%) — útil pra Pareto
    export function withAlpha(hex: string, alpha: string): string {
      return hex + alpha
    }
    ```

    Avoid: depender de hash criptográfico ou lib externa. Heurísticas básicas + fallback determinístico bastam.
